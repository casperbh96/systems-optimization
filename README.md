How to run:

1. Install Python interpreter with Python 3.8 and Pip 3
    a. Miniconda:

    b. PyCharm:
        1. Press Terminal on the bottom OR View > Tool Windows > Terminal
2. Install requirements.txt using: pip install -r requirements.txt
3. Run the main file, e.g. by a console command: python main.py
